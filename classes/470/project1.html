project 1 blog post

Background

For Project 1 in IGME 470 we had to create some sort of an interface to collect or interact with a plant.

Sensors?

I had a sensor kit that has been sitting in a little box in the corner of my room for years at this point, and when thinking of ways to interact properly with a plant I figured I could use some of the sensors that I already had on hand.

Original Plan

My initial plan for project 1 was to use an array of sensors and an oled display to display relevant information for the user- and have it be situated by the plant to give a HUD style display to see what is going on with the plant- With a soil sensor and an ultrasonic sensor so you could “say hi” to your plant.


Circuit Diagram

Picking a Sensor

After assessing the sensor that I already had on hand, I was incredibly interested to delve more into the DHT11 temperature / humidity sensor that I had. It was very interesting to read up on the sensor and realize that is uses a serial communication between the sensor and the mainboard. This would allow for both the temperature and the humidity information to be sent over one output pin.

Control Inputs

With the sensor input being finalized, I wanted to figure out a better way to control and add a level of “setup” to the sensor. So I decided to program in a button system to edit and modify the desired humidity and temperature. This would allow you to optimize a room to best suit the plant that you are trying to keep alive.

Parts

  • Piezo
  • Arduino Uno
  • Potentiometer
  • Button
  • 10K Resistor
  • DHT11
The Final Circuit on a Breadboard.

Functionality

Two presses on the button would allow for the user to enter a settings setup mode, and there are piezo beeps to remind you that the process is active and the button was pressed. The potentiometer would then allow you to select the desired values +/- 10. After exiting settings mode, if the room ambient temperature or humidity was above or below the desired values, the piezo would beep (eerily similar to hospital monitoring sounds).
